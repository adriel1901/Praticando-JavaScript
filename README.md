### Tabela da Classe Math
| Função                | Descrição                                                                                            | Exemplo                               |
| --------------------- | ---------------------------------------------------------------------------------------------------- | ------------------------------------- |
| `Math.abs(num)`       | Retorna o valor absoluto de um número. Se o valor for negativo, ele será convertido para positivo.   | `Math.abs(-3) => 3`                   |
| `Math.ceil(num)`      | Arredonda o valor para cima, retornando o próximo número inteiro.                                    | `Math.ceil(4.2) => 5`                 |
| `Math.floor(num)`     | Arredonda o valor para baixo, retornando apenas a parte inteira do número.                           | `Math.floor(7.9) => 7`                |
| `Math.pow(base, exp)` | Retorna a base elevada ao expoente informado.                                                        | `Math.pow(3, 2) => 9`                 |
| `Math.random()`       | Retorna um número aleatório entre `0` e `1`, com várias casas decimais.                              | `Math.random() => 0.6501314074022906` |
| `Math.round(num)`     | Arredonda para o inteiro mais próximo. Valores com decimal `.5` ou maior são arredondados para cima. | `Math.round(2.7) => 3`                |
| `Math.sqrt(num)`      | Retorna a raiz quadrada do número.                                                                   | `Math.sqrt(16) => 4`                  |
